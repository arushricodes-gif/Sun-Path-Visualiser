# Sun-Path-Visualiser
This interactive Sun Path Visualizer tracks solar movement and environmental data for any location. It features a custom Leaflet animation showing sun trajectories, real-time AQI, Temp, and Wind metrics via the IQAir API, and an elevation chart. Perfect for architects or gardeners to analyze shadows and light throughout the year.
